# @amlplugins/sendgrid-domain-authentication

Native `@sendgrid/client` SDK re-exported under the `@amlplugins` namespace.

> SendGrid Domain Authentication — SPF/DKIM/DMARC, sender authentication, link branding.

## Install

```bash
npm install @amlplugins/sendgrid-domain-authentication
```

## Use

```ts
import { /* named exports from @sendgrid/client */ } from "@amlplugins/sendgrid-domain-authentication";
// or
import sdk from "@amlplugins/sendgrid-domain-authentication";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
